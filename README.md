# 🎬 Book My Show Clone (MERN Stack)

A full-stack movie ticket booking web application built using the MERN stack.

---

# 🚀 Features

- Select movies
- Choose slots/timings
- Book seats
- Store booking data in MongoDB
- Display latest booking details
- React Context API state management
- REST API integration

---

# 🛠 Tech Stack

## Frontend
- React.js
- CSS
- Context API

## Backend
- Node.js
- Express.js

## Database
- MongoDB
- Mongoose

---

# 📂 Project Structure

book-my-show/
│
├── front-end/
│ └── React Frontend
│
├── back-end/
│ └── Express + MongoDB Backend

---

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/book-my-show.git
```

## Frontend Setup

```bash
cd front-end
npm install
npm start
```

## Backend Setup

```bash
cd back-end
npm install
npm start
```
